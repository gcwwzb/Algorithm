[语法分析器]    
======
<br>
   将“词法分析”的输出结果，作为表达式语法分析器的输入，进行语法解析，对于语法正确的表达式，报告“语法正确”；<br>
　　对于语法错误的表达式，报告“语法错误”， 指出错误原因。<br>
   把语法分析器设计成一个独立一遍的过程。<br>
   语法分析器的编写方法采用递归子程序法。<br>
