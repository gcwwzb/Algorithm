[处理机调度算法]    
======
<br>
  模拟处理机调度<br>
  并实现了图形界面<br>
  多道系统中，当就绪进程数大于处理机数时，须按照某种策略决定哪些进程优先占用处理机。<br>
  本实验模拟实现处理机调度，以加深了解处理机调度的工作。<br>
<br>
<br>
<br>
  包括优先权调度和时间片轮转两种方法<br>
