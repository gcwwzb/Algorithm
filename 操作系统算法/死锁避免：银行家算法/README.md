[银行家算法]    
======
<br>
  多个进程动态地共享系统的资源时可能会产生死锁现象。<br>
  银行家算法是通过对资源的分配进行动态地检查来达到避免死锁的目的。<br>
  本实验通过模拟银行家算法的应用，使读者了解银行家算法的执行过程。<br>
  从而进一步理解死锁产生的条件和避免死锁问题产生的方法。<br>
